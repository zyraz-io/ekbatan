# Ekbatan Agent Skills

This directory is the source of truth for portable Ekbatan agent skills.

The same skill folders are mirrored into:

- `.claude/skills/ekbatan-*` for Claude Code repository discovery.
- `.agents/skills/ekbatan-*` for Codex repository discovery.
- `plugins/ekbatan/skills/ekbatan-*` for Claude plugin publication.

After editing a skill here, run:

```bash
agent-skills/scripts/sync.sh
```
