The SQL reference now lives at [Framework tables](/reference/tables/), because the schema reference covers more than the base event table:

- [`eventlog.events`](/reference/tables/events/) — committed action/event rows written by the default event persister.
- [`eventlog.event_notifications`](/reference/tables/event-notifications/) — optional local-event-handler delivery rows.
- [`scheduled_tasks`](/reference/tables/scheduled-tasks/) — optional db-scheduler table used by distributed jobs and local handlers.

Keep using this URL for old links, but use [Framework tables](/reference/tables/) for new references.
